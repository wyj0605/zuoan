# 我的WordPress插件之旅 

WordPress发展到2.6.3，很多以前要用插件实现的功能WP后台已经具备，但不可否认的是WP插件让Wordpress更具生命力，也是因为WP Plugin，Wordpress的开源才更有意义。插件可以使我们的WordPress更具灵活、更易用、更独特、更亲和。下面介绍我喜欢的WordPress插件：
中文 WordPress 工具箱 我想几乎所有中文Wordpress用户都会用到的插件吧，提供了很多实用功能，比如显示随机文章，最新留言（最新引用），留言最多文章，发表评论最多的网友，还有真正的文章摘要。而且定制性也比较强
©Feed  适用于WordPress的©Feed插件功能强大，可在feed footer添加版权声明、数字指纹、文章留言、相关文章标题，还可以控制feed是否全文输出，因而很多bloggers都安装并启用了这款插件。
Akismet WordPress官方推出的一项反垃圾评论服务，以前我也用过其他各种办法，比如评论必须有一个中文字，但Akismet很不错。
All in One SEO Pack 让你的博客更适合Google等搜索引擎的口味，这款插件是必不可少的。可以为你的日志自动生成Meta标签，修改Title，添加描述语句等。——不过水煮鱼先生建议大家别用All in One SEO Pack
Google XML 对于那些想Google更好的索引，或者在用Google网站管理员工具的用户这款插件是必备的，为你的博客生成一张Sitemap.xml页面，可以对博客内容每部分进行抓取频率设置。
No Self Pings 这个插件可以让你的Wordpress忽略你在文章中对自己博客日志的引用。
Share This 中文 很强大的分享插件，集成了几乎所有Web2.0分享功能。一键分享到Delicious，Facebook，Twitter等，而且还可以通过修改php文件来添加更多的分享模块。
Ultimate Google Analytics 我一直用的插件，只需要填上Google Analytics的识别码就可以用上这款我认为目前最强大的免费浏览分析服务了。听说google广告后台可以直接申请这个分析模块，那就太好了。
Simple Tags ：Simple Tags绝对是目前为止最为好用的wordpress tag插件，可以说它汇聚了以往tag相关插件的种种优点，集百家之长于一身，是目前绝对不可以缺少的wordpress插件之一。SEO必备插件
WP-PageNavi 摆脱页面浏览单调的“上一页”“下一页”。为多页面内容添加一个更友好和人性的页面导航。
Ajax Post Rating ：wordpress文章评分插件，可以让读者给每篇文章评分。另外可以自定义输出评论最多的文章列表。
当然插件是越少越好，不然你会感觉你的WP就如一只大笨象，会拖累你的服务器！
