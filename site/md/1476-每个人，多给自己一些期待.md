# 每个人，多给自己一些期待 

今天有三个段落，因为它们都和那个美国日本有点关系，我就把他们放在一起，供大家思考。
第一个是让我很敬重的时寒冰先生的一段话： 
“公平公正的制度才能让人民免于掠夺和恐惧，才能让公心和敬畏生命成为普遍的价值观。”这是我博客的主旨。但是，所有美好的制度，都是民众推动的结果，而不是所谓的英雄或救世主主导的结果。
我始终相信，有什么样的人民，就有什么样的制度。就如同西方“有什么样的人民，就有什么样的政府”的理念。
如果希望寄托在所谓的英雄身上，只会沦为一场空。个体的觉醒，每个人的共同努力，才能真正推动社会进步。
中国，源于民族深处的劣根性，是一种恐怖而邪恶的力量。
其实，好的环境、好的制度，都是每个人努力的结果。一个最直观的例子：许多人都会谴责乱丢垃圾的行为，但是，扪心自问，自己是否能够做到不乱丢垃圾，进一步，是否能做到，把随手可以捡到的垃圾放到垃圾箱中？
日本大和民族，在一场盛大的体育运动会后，不留一片纸的惊人一幕，让人感受到一个民族可怕的力量，我们具备这种素质吗？
改变现状，不是寄希望于别人，不是把所有的期待都被别人，它需要每一个人多给自己一些期待，多给自己一些要求。
我们都是普通人，这世上其实并没有英雄，所谓的英雄都是真实原型加创造的结果。但如果把英雄定义为做得比一般人要好的人，那么，人人其实都可以成为英雄，而到了那种境界，还是不再有英雄。
我的意思是，永远不要把希望寄托在别人身上。
这个世界是由一个个具体的个体构成的。
如果某个人被寄予太多希望，无异于一种扼杀。
我所做的研究是趋势研究，是前瞻性的研究，来不得半点虚假，因为，一切的一切，都将很快被验证。我在去年年底写的《中国怎么办》，提出通货膨胀隐忧、资源 为王观点时，这个世界还在饱受紧缩之苦和大宗商品价格大幅下跌的现状，而在已经进入2009年6月份的今天，恐怕已经鲜有人怀疑我当初的判断。趋势研究是 最费心血的，郎咸平先生说，我做的应该是一个拥有至少几十人的团队才能完成的工作量，有时候几乎要累倒。通过讲座（比如像春节前在郑州举行的十几个人参加 的微型讲座），发布对趋势的研究结论，让听讲者一起见证结局，分享成果，是我所喜欢的。事实上，这样的讲座，听者是最大的受益者。比如，春节前我在郑州讲 座中讲解的股票至今都有两倍以上的利润。我对讲座的态度是认真的。上次在北京知钱俱乐部讲解大宗商品价格走势，我获悉春节听过我讲座的三位朋友也参加，马 上把10%的重叠部分全部更换了，不让支持我的朋友听重复的内容，哪怕有一位这样的朋友在场。
我非常喜欢趋势研究。喜欢才能投入。结果的验证常常是让人欣慰的。2008年6月，我接受央视采访时，认为油价将大跌（当时的油价是每桶140美元左 右），2009年3月，我提出“如 果3月、4月贷款继续增长，国际油价可能突破60美元，到达60美元至70美元的区间。如果这个期间俄罗斯对美国没有什么过激行为【以俄罗斯领导人的智 慧，他们会韬光养晦，绝对不会打肿脸充胖子】，而中国经济脆弱的一面越来越明显，美国可能快速将油价拉到70美元上方，甚至再次突破100美元（当时的油 价是每桶30多美元）。而现在，猪流感的到来，再次埋下了一个可怕的伏笔，加快了油价上涨之路”（——摘引自本人写的长篇分析《从房价到经济，地狱亦或天 堂——中国经济发展趋势的反思之一》）
有什么样的因，必有什么样的果。不停的努力，使得利益分析法逐渐完善，5月21日写的《美国孤注一掷，最坏时刻还没来》一文末尾，我提到“补充：关于股 市，如果政府真要启动IPO，只会继续上拉股市，直到绝大部分人轻视IPO的负面影响时为止。”这也是利益分析法推导的结果。
我不知道为何要说这些，并非单纯为自己辩解，而是讲一个道理：每个人，都应该多给自己一些期待，而不是苛求别人。社会的进步，是群体觉醒和努力推动的结果。
苛求别人不如多苛求自己，多给自己一些期许。即使有一天，由于种种原因不再写作或者做研究，我仍然会为大家祝福！
第二个是我认为兼具冒险与理性思考的占豪先生对中国未来的一点思考：
关于中国，很多国人由于多年的自卑心理作祟，一直无法自信，每每都把西方的东西给深化而贬低自己，这种妄自菲薄之人不是被收买就属于那种过度自卑者。中国 的走强不是某个人决定的，这是时代赋予的机会，就像谁能够想到一战后英国会逐渐走下神坛，走向没落。对美国来说南北战争后才走向发展的正轨，当时大英帝国 还处于鼎盛时期，绝不亚于今天之美国，而当时的美国却不如当今之中国。世界变幻，这是由时代决定的。从战略意义上说，中国只要不犯大的错误，抓住未来二十 年的机会，中国必然能够引领世界。但这里边存在的风险就是中国不能抓住这二十年机会发展自己，而美国经过大调整后再度控制世界，一旦如此中国将至少再有一 百年以上甚至数百年不可能达到世界顶级。这种时候，每一个中国人都应该为了自己的子孙后代做出自己的贡献，否则有何颜面面对自己的子孙，告诉他是因为自己 这一代没有努力而使他们失去了发展的机遇？恐怕到时就无言以对了。
第三个是信力建先生写的关于“美国人究竟对日本做了些什么”的文章：
美国改造了日本什么？
1945年8月15日，日本宣布无条件投降。麦克阿瑟为执行美国政府“单独占领日本”的政策，以驻日盟军总司令名义，在东京建立盟军最高司令官总司令部。 为了确保日本不再成为美国的军事威胁，美国政府开始了长达7年的重建日本计划，以美国政治经济制度为模式，对日本的政治、经济、军事和社会各方面实施了大 规模改革。这种改革被很多人戏称为美国人对日本的报复、惩罚。其实是日本真正走向现代化道路的开始，当然是美国主导的、美国政府成为日本“太上皇”为前提 的改革。改革的最基本依据和政策指导是美国政府颁布的两个文件，1945年8月29日批准的《美国战后初期对日政策》和同年11月3日发给麦克阿瑟的《有 关投降初期占领和管制日本的基本指令》。那美国究竟改造日本什么了？我们始终认为，美国占领日本，乃至美国作为盟军首领之一占领战败国是世界历史上精心计 划的，借外来势力指导的大规模政治改革行动，全球推销美国的政治制度。
1. 修改宪法，改革日本的政治制度，实行政教分离和废除天皇制度。
美国从1945年着手日本各项民主改革，并通过立宪的形式彻底将美国的政治制度植根于日本。 由于麦克阿瑟与日本政府在宪法中如何规定天皇的地位问题上存在严重的分歧，遂抛开日本政府，美国负责修改宪法。根据最后出炉的“和平宪法”，（1）废除了 天皇，但天皇还是国家名义上的主人，也只能扮演纯粹仪式上的角色；（2）废除有强烈封建色彩的政府机构，如枢密院、内大臣府等天皇的直属机构，建立带有英 国议会制色彩的政治制度；（3）放弃发动战争的权利，但拥有自卫权，也不能违反联合国“集体自卫”的禁止规定。除了这些特别的条款外，日本和平宪法很大程 度上与美国宪法保持一致。这也容易理解，和平宪法草案就是有麦克阿瑟的两个负责法律事务的下属军官负责撰写。尤其要指出的是，和平宪法也将“政教分离”条 款写入其中。战后日本政府宣布政教分离，裕仁天皇发布诏书，宣布自己是人不是神，废除国家神道，政府不得资助神社。关于日本神道教，后面将详细阐述。
2. 社会民主方面的改革，撤销新闻限制，解除党禁和报禁，促进工会建设。
麦克阿瑟改革的最初切入点是推行民主和自由。对日本国民进行广泛的意识形态改造是美国 重建日本的重要部分。1945年9月22日公布的《美国战后初期对日政策》中就提出要铲除日本人心目中的军国主义和极端民族主义思想，鼓励日本人民培养起 争取个人自由，尊重基本人权，特别是宗教、集会、言论和出版自由的愿望。9月10日，麦克阿瑟对日本政府下达了关于言论和新闻自由的命令，时隔半个月的 26日，又下达了取消一切通信自由的限制，鼓励日本人民批评政府的政策。10月4日，麦克阿瑟颁布了题为《撤销对于政治自由和其他自由的限制》的命令，废 除了日本所有限制基本人权的法律和法案，这项命令因此而成了日本人的“人权法案”。同时也发布了“废除对政治、民权及信仰自由的限制”法令，并释放了全部 在押的政治犯。麦克阿瑟对日本首相币原喜重郎的“确保人权的五大改革”是促使日本社会走向西方民主化道路的重要措施，其中包括男女完全平等、促进工人的团 结和组织、教育的自由主义化、废除专制政治、促进经济民主化。解除党禁和宗教信仰压制，各种政党竞相成立，其中包括共产党、自由党、进步党等，各种宗教团 体也纷纷出笼。
3. 废除财阀和农地改革。
解散财阀，实现经济生活民主化。农地改革，摧毁日本经济的封建主义基础。财阀集团是战前日本最有势力的经济力量，实际控制和操纵日本 的政治、经济发展方向。正是这些财阀的存在，日本的一切征服和侵略行为成为可能。1945年9月21日，麦克阿瑟发布解散财阀方针的指令，要求日本政府消 除经济生活的垄断现象，鼓励并赞助在民主基础上组织起来的工业、农业和劳工组织。随后，冻结了46家具有军系色彩的财阀组织的财产；解散了83家大型财阀 企业。1947年，日本政府分别颁布以限制与分割垄断企业为目标的《禁止垄断法》和以实现经济民主化为目标的《经济力量过度集中排除法》，先后解散了三 井、三菱、住友等最大的财阀集团，限令财阀家庭的领导一律辞去其担任的经济职务。但是由于后来爆发的冷战以及日本政府的敷衍塞责，财阀解散政策执行并不彻 底并留下祸根。
战前日本的土地制度被麦克阿瑟斥责为“实质上的奴隶制”，严重阻碍了农村经济的发展和造成了农村阶级矛盾的升级恶化。1946年，在麦克阿瑟的干预下，日 本国会通过了两项土地税改革法案，彻底完成了清除封建土地制度的历史使命，打破了使农民处于奴隶化地位的经济桎梏，为日本经济的重建确立了新的基础。后来 日本农业获得空前的高产，麦克阿瑟主导的土地改革功不可没。
4. 去军事化和处决日本战犯。
麦克阿瑟解散了日本多达589万的军队和不计其数的准军事组织，实现了去军事化，这是民主化政策能够实现的最大前提。1945年9月起就开始了解散日本军 队，实现非武装化，废除了《兵役法》。到10月15日，日本全国武装力量全部得到解除，日本军队不复存在。1947年，日本在国外的300多万的军队陆续 被遣返回国，作为军事强国的日本从此消失。对武装力量进行解散的同时，也对一切鼓励军国主义、赞成或助长日本建立亚洲霸权的政党、组织进行取缔、整肃。从 1946年初开始到1948年3月结束，整肃持续了整整两年。期间取缔了黑龙会等27个极端民族主义政党组织，后增加到147个。
此外，与处理纳粹战犯一样，盟军处治了日本战犯。1945年9月11日，逮捕了前首相东条英机等首批39名甲级战犯，年底又逮捕了104名甲级战犯。 1946年1月，远东国际军事法庭成立并开始审判战犯，1948年11月，法庭判决首批25名战犯中7人绞刑、16人无期徒刑；惩处其他军国主义分子20 万人。同时，美国对日本执行“战争赔偿计划”和偿还被占领领土。中国出于各种原因没有向日本索取战争赔款。领土问题在盟军的干预下也未能划清，如钓鱼岛、 台湾等，为日后的冲突埋下了祸根。
失败的政教分离？ 
神道教简称神道，是日本的传统民族宗教。它和日本本身一样古老，在日本是最重要的宗教。在日本大约有1亿600万的人信奉神道教，大约占日本人口的 85%。为了与汉传佛教分庭抗礼，就创造了“神道”一词。神道教与佛教的地位争夺此起彼伏，经过很长一段时期才达到相对的均衡，出现了佛教寺院与神道的神 社两者并存的现象，但供奉天皇祖先的伊势神宫则例外，只属于古神社。至明治初年，神道成为国家的宗教，日本正式走向政教合一的国家，崇拜神道成为日本国民 的义务，直至二战结束日本投降后，在盟军的要求下，宣布政教分离。
日语中的“神道”意思与汉语中的不同，日本人将一切神明用日语称为Kami，汉字“神”就被用来表示Kami。Kami包括皇室、氏族的祖先、已逝的人之 灵魂。此外还将大自然中被认为值得崇拜的山岳、树木、狐狸等动植物的灵也称为Kami，还包括一些骇人听闻的凶神恶煞也称为Kami。由此，可看出神道教 是自然崇拜、祖先崇拜、天皇崇拜，属于泛灵多神信仰，因此日本人认为有八百多万神也不足为奇了。在神道教里，神是平等的，神没有等级之分。将普通人的灵 魂、天皇的灵魂和花花草草的灵魂都视为神，也代表着神没有高低贵贱好坏之分。供奉Kami的地方被称为神社。神社一般不设立灵牌，一些具有重要任务的神社 被称为神宫（通常与供奉日本天皇的祖先有关）。神社和神宫也是日本人祭祀的主要地方。
在日本，比较重要的几座神社中就包括了靖国神社。靖国神社的名字取自《左传》的“吾以靖国也”，意为使国家安定。靖国神社的前身是建于1869年的东京招 魂社，1979年改名为靖国神社。迄今为止它已经有140年的历史了。截止至2006年，靖国神社祭祀着246.65万在戊辰战争以来历次战争中为日本政 府战死的人，包括军人和军属，其中有210万死于二战，包括14名二战甲级战犯和约2000名的乙、丙级战犯。根据靖国神社的官方文献，靖国神社的祭祀名 单包括阵亡的军人、战场救护的医务人员和女学生、在军需工厂中死亡的学生、有出身台湾的日本军人、有出身朝鲜的日本军人等，靖国神社都将他们一视同仁尊称 为“靖国大神”。
除了祭祀军人和军属让靖国神社影响力非常之外，它在日本战败前都是由军方管理控制的渊源也不可忽视。而其他神社则由国家内务省管理，靖国神社因此显得独树 一帜。在二战盟国的强烈要求下，日本战后的和平宪法表明政教分离，国家不可以介入任何宗教事务。因此现在靖国神社变成了一个非政府的宗教机构（宗教法 人），近似商业机构。不可否认的是，军方的长期管理和供奉军人的灵位，这些都很难让人不去怀疑靖国神社的“军国主义色彩”，也让靖国神社超出了作为神道教 的祭祀机构的宗教范围。
日本政府官员经常以官员身份参拜也是让靖国神社“闻名全球”的很重要原因。这里我们要认识到最重要的一点是，日本人若以官员身份参拜神社，不管是靖国神社还是其他庙宇，都是违法宪法，触犯了日本宪法有关政教分离的条款。
宗教向来是一个国家的内政，不干涉宗教事务是目前世界上绝大多数政府的共识。但宗教活动必须在法律范围内活动，不得反人类等。祭祀亡灵一直以来是很多宗教 的最重要内容之一，如我们的佛教、道教等，甚至是无关宗教的日常生活的主要内容之一。后者则属于道德行为范畴。日本法西斯侵略了亚洲很多国家，生灵涂炭， 是严重的反人类罪行。若把这些战犯作为“国家的功臣”或者崇拜的偶像进行祭祀，这肯定是受害者不能接受的事情。若把它作为一个国家的国民、一个普通人、一 个普通家庭的祖先进行祭祀，这又该如何呢？一将功成万骨枯。胜者为王，败者为寇。只要是触犯人类罪行的人，都不能作为崇拜的偶像进行祭祀，正如希特勒。同 时，那些很多胜利者的典型，是不是也该那样呢？仁者见仁，智者见智，仿佛陷入了康德的二律背反悖论里面。
和很多一神论宗教不同的是，神道教并没有绝对的理念，没有绝对正确和错误，也没有完人。神道它是一种乐观的信仰，人被认为本质上是好的，邪恶被认为是由于 邪恶的精神而引起。所以，大多数神道教仪式的目的就是通过净身（祭祀前的洗手、漱口仪式）、对神的祷告和献祭而驱除邪恶的精神。
后记：日本停滞发展10年？笑话！ 
据媒体报道，2007年底，日本财阀三井公司在巴西购买了10万公顷农田，这一耕地面积相当于日本本土的耕地面积的2%之多。根据非政府组织GRAIN在 2008年公布的一份报告，日本的食品企业于2006年到2008年间在巴西、非洲和中亚等地租用和购买了大量农田以种植有机作物，日本目前已拥有超过国 内农田3倍的海外农田。
现在我们只要在google中搜索“日本经济”关键字，看到的信息都是有关日本经济崩溃、停滞、黑暗、深度衰退等等的报道。可能我们从表面数据看到了日本 经济正在走下坡路，更准确地说应该是调整期。可事实上是怎样呢？经历了战后50年代二位数的飞速发展，在世界经济体排行榜中占据第二把交椅近半个世纪的日 本，目前的经济发展数据虽然不太快速，但也不至于衰退。正如一个人在青春期是疯了一样长个子，到了成年就不是再长个子，该变成熟、变稳重。换而言之，是长 智慧的时期。目前的日本就是如此。我们从前面的媒体报道可以读到这点，这是日本人在长期的危机意识中迸发出的智慧，是自我拯救。
日本以前是以贸易立国，主要靠商品原料的进口和成品的输出为发展模式。在20世纪大部分时间里，全世界的人们都希望能买到日本产的产品，尤其是电器产品。 由于成本等条件的限制，日本就开始直接在海外设厂。如果说20世纪的日本是在全球设立工厂进行“播种”的话，那么现在是“收获期”。日本企业全面走出去投 资全球，或设厂、或控股、或金融市场、或贷款给其他国家等方式，是以投资为导向的发展模式，即是“投资立国”。如今的日本，不光在耕地方面在海外重新“构 建”了一个日本，更是在资产方面在海外重新“建立”起一个日本。
日本的成熟方面，表现在科技研发能力的全球领先地位、环境保护能力卓越、国民素质全球数一数二等几方面。日本非常成熟的经济发展模式，又怎么会走向衰绝 呢？只能说中国有些人被假象蒙蔽了，非常简单愚昧地认为日本停止了发展，这对我们确定对日本的政策有非常大的影响。2003年的统计，一个东京的GDP， 相当于20个上海，10个汉城，7个香港，5个巴黎，3.5个伦敦；中国的GDP是1.4万亿美元，日本是4.3万亿美金，日本是中国的三倍。目前我国的 经济形势好了不少，2007年超过了德国成为全球第四，而2008年的数据显示出中国与日本的差距缩小了很多，于是就有些学者和机构在叫喧：2009年中 国GDP赶上日本。但现在日本领先我们的方面实在太多，光一个GDP的超越又能说明多少问题呢？只能说一些中国的学者实在是夜郎自大的要紧。
