# 蜡笔小新的星座语录 

先说点题外话：在阿禅日记里得知《蜡笔小新》之父去世的消息（在日新月异的信息社会里，这算是旧闻了），有点不敢相信，难道创作者不可以来一次如漫画般的重生，或者这只是小新般的一个“恶作剧”，但无论真相是什么，“小新”都是曾经给我们带来欢乐的人。
《蜡笔小新》实际上不是一部小朋友可以直接看的动画，我不知道为什么它被引进中国后就被认为是一部小朋友动漫，或许在传统的眼光里，动漫就是给小朋友看  的。但是，香港播放《蜡笔小新》的时间多是晚上10点之后，并打上了“家长指引”的标记，意思是没有家长的指引，小朋友不要看这个节目。
现在我会让孩子看看中国版的小新——《大耳朵图图》，至于看《蜡笔小新》，我是一定要陪着孩子一起看的。
好了，我们要说什么来着，对了，是要想说说小新是什么星座的？小新有什么可爱搞笑的星座语录呢？我们一起来看看吧~
白羊座：
心直口快的小新 /  爆笑指数★★★★
小新：有酱油卖吗？
鱼铺老板：没有。
小新：有芥末卖吗？
鱼铺老板：没有。
小新：什么都没有还敢开店。
金牛座：
会精打细算的小新  /爆笑指数★★★
美芽：“小新，你这学期捡到10次钱吗？”
小新：“没有，只捡到一次”
美芽：“那怎么会有十张拾金不昧的荣誉卡？”
小新：“我把捡到的一百元换成10个铜板”
双子座：
能言善辩的双面小新  /爆笑指数★★★★
(饮料机旁)
小新：妈妈，妈妈！
美芽：我是不会给你买的！
小新：平时家里穷，买不起，难道出来玩的时候都不能给我喝一杯？
美芽：你这个孩子，平时天天喝……
(众人围观，美芽只好买)
巨蟹座：
很会照顾人的小新  /爆笑指数★★★
管理员：这里不能带狗进去，我先替你看着这条狗吧。
小新：虽然它长得象棉花糖，但是你也不能把它吃掉哦！
狮子座：
超级酷而有自信的小新  /爆笑指数★★★
风间：你这样跳舞是不对的。
小新：你不要崇拜我。
风间：我没有！
小新：你不要不好意思。
风间：我也没有！
处女座：
咬文嚼字似乎很较真的小新  /爆笑指数★★★★
美芽：以后吃饭的时候不能看电视！
小新：绝对吗？
美芽：绝对！
小新：绝对的绝对吗？
美芽：绝对的绝对！
小新：你还真孩子气……
天秤座：
颇为讲究公平的小新  /爆笑指数★★★★★
美芽：“上完暑期辅导课就要马上回家，为什么不听话？我讨厌不准时回家的人
小新：“你干嘛那么生气？你下班没马上回家作饭，我也没生气啊！”
天蝎座：
深喑抓人把柄的小新  /暴笑指数★★★★
美芽：你就是撒娇我也不会帮你买那样东西的。
小新：你不帮我买我就大声喊你内裤的颜色。
射手座：
追女生搭讪高手的小新 /  爆笑指数★★★★
小新：小姐，你家住的远吗，你喜欢吃纳豆吗？下次能赏光坐我的三轮车吗？
摩羯座：
惜时如金的小新  /爆笑指数★★★★★
小新：“老师，我要上厕所”
老师：“不行，现在是上课时间，刚才下课怎么不去？”
小新：“下课时间那么宝贵，用来上厕所多可惜呀！”
水瓶座：
超级搞怪逻辑的小新  /爆笑指数★★★★★
美芽：“小新，你又开电视了”
小新：“我又不是要看电视”
美芽：“那你在做什么？”
小新：“我在核对报纸上的电视节目表有没有印错”
双鱼座：
充满同情心的小新  /爆笑指数★★★★
小新：“美芽，公园有个可怜的欧巴桑，我想帮助她”
美芽：“小新真有爱心，？给她10块钱吧！”
美芽：“咦！你怎么买了香肠？”
小新：“她就是卖香肠的嘛！”
