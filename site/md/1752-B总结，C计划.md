# B总结，C计划 

去年的这个时候，我写了一篇A总结，B计划，总算给自己忙忙碌碌的生活有个交待。现在2008也要过去了，一样的不能例外，就来个B总结，C计划吧！如何做总结和计划，依然可以参考弥缝的这篇文章！
依然先报喜了：
儿子成长得很好，他最喜欢在地上快步小跑，常常在外婆家追着小狗到处闪躲，只是已经14个月了还不肯叫“爸爸、妈妈”，看来，教育孩子是人生最大的一项工程了，所以，我在百度为他建了一个成长博客，由我们家的“一把手”记录孩子的成长和一些育儿经验。
家人今年身体健康，偶尔小病，都进行了及时预防。家人关系和谐，一家人在一起就应该融融洽洽！爷爷已经80岁了，身体硬朗，生活乐观。小弟最近的工作也稳定下来，做爸爸就是要有做爸爸的样子。小外甥女已经能很流利地说话了，可以整天和她斗乐！
我想，男人最好把女人的啰嗦当幸福。老婆会把生活安排得井然有序，自然也会说这个不对，那个不好，我最好依从，小两口伴伴嘴，也算过得开开心心。
今年终于评了一级职称，算是意外的收获吧！
去年制定的左岸3500的订阅量的目标算是超额完成了，我知道这里面有很多的水份，但7000多的订阅数还是让我有些开心，这里面多多少少反应了长尾的效应。
这一年，左岸走得虽然不是很顺利，中途还退出了博客圈3个月，但最终还是坚持了下来，还成就了两个新左岸，一个我的，一个胸不平姑娘的。
反思一下：
这一年有很多的遗憾，这我喜欢的几个网站好像遇到了什么问题。抓虾几乎一年没有新动作，feedsky的话题也沉寂了许久，鲜果也没有再突破了，阿里妈妈也没有什么进展，keso很少写文章，麦田已消失，月光很琐碎，和菜头又常恶搞……太奇怪了！
今年最大的损失又是股市，去年是说了大话了，结果不是往上翻，而是往下翻。终于明白，做事不是仅凭一腔的热情就可以的，那需要很多的智慧和经验，需要在大量的信息中看出事情的本质和发展方向，方向错了，做什么都白搭。
今年感觉自己脾气有点大，没有很好的控制自己的情绪，这使得身边的人常常受到伤害。其实生活可以简单些，应该要有豁然心胸，能容天地万物。
简单计划一下：
给左岸一个新目标，1万的订阅量，分享更多精彩文章。
设计一些作品，至少给左岸设计一款自己的主题（向幸福老大学习）。
找寻新朋友，主动与朋友交流，共同探讨人生问题。（喜欢古兄、兰花兄这样的朋友）
完善投资知识，学会放眼看天下，洞察人世百态。
给家人更多照顾，做好时间安排。
禁止熬夜。
