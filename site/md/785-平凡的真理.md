# 平凡的真理 

我们身边朋友的一席话往往比长篇的大道理对我们更加有效，因为来自同一界面的对话令人感到更加真实，仿佛那就是对我们说的。这篇文章是sly61的体会之作，读来就像读《牛奶可乐经济学》一样有趣。
1. 关于理想、欲望与幸福~
几乎所有孩子都知道，奥特曼的战斗形态最多只能维持三分钟，时间将到，身上的警报器就会响起，这时候，无论打没打死怪兽都要闪人，否则迎接这些宇宙英雄的就是死，因此奥特曼的维护宇宙和平的理想，实际上就是一个三分钟接连着另一个三分钟，一个小怪兽接着另一个小怪兽。
这里面有一些比较深的暗喻和嘲讽，那就是大人们常说的三分钟热血。再艰深的欲望，不，应该说再强大的能量维持的时间都不会很长，唯有在极短的时间内让它发挥作用才可以达到最好的结果，这就跟奥特曼必须在三分钟内打倒小怪兽是一样一样的。
如果你月工资一千七，你想买一台两千元的PSP，那这台PSP无疑是可以作为理想的，因为当你得到PSP的时候你可能会很幸福，而且这目标离你的努力也相去不远，它会让你自信满满同时迫不及待完成这个月的工作，这时候的理想是有益的，积极的。但如果在同样的基础上，你把理想定位在一万三千元的苹果笔记本  上，那这个理想很有可能会在你付出所有努力之前就灰飞烟灭，原因很简单，可能仅仅是厌倦或者疲惫。
（所以，我不甘愿连小怪兽都没有打过一只的活着！）
2. 关于征求意见~
你向别人询问意见的时候，直接问怎样做，要远比问“可不可以”实际得多，如果你是一个新手请切记这个忠告。
一、在询问意见时，要清楚地分辩意见和劝诫之间的区别。
二、如果已定好了方向，那么对智囊团的反馈信息，就应接受建议，摒弃劝诫。
三、要抱着“必胜的决心”和“必败的准备”，随时为自己的选择负责。
如果具备以上三点觉悟，你心里可以对试图阻挠你的计划的人说：思想有多远，你就给我滚多远。
3. 关于朋友~
被出卖的大多都是兄弟，被利用的永远都是朋友。
安妮宝贝说：人性的成熟不是越来越包容，越来越涵盖，相反却是一个逐渐剔除的过程，知道自己需要什么，可以信任谁。
向一个朋友要求一件可能麻烦他或可能不喜欢的是事是不礼貌的。
我很懒，所以很讨厌别人麻烦我。心情好时除外！
4. 关于职场生存~
对绝大多数的主管或者总裁来说，他们需要的不是有太多想法的人才，而是可以最大程度上完成自己想法的员工。他们心中的好员工的形象，应该是一颗拧到哪儿都可以起到作用的螺丝钉。
5. 关于教育~
古人云：教，上所施，下所效也；育，养子使为善也。
从上面的字句往深里挖掘，无论是上行下效，还是养子为善，中国教育的精髓就是做人。
6. 关于设计、提案~（因为我也是设计，所以深有同感。括号部分是我的感想）
记得以前做设计的时候，经常发生这样的事情，接到一个设计单，然后委托人要你定期拿出方案，你问他有什么想法的时候，他往往说不出什么成型的意见，要你自由发挥。你把设计方案的交上去的时候，委托人一反当时没有主心骨的态度，马上变得诸多意见，让你这个照着他的想法改，那个要你按照他说的详细做一下，忙   活了大半天，照着他想法做出来的方案，连他自己都看不上，结果到头来还是采用你最初的那个方案。（另：还有一种客户总是喜欢用“不大气”这个词，这个词及   说这个词的人也是很能激发人类暴力倾向的。）这是个没有权威或者人人都是权威的时代，好多人都喜欢拿别人的汗水来实践自己的想法，这是一种廉价而又快捷的成功手段。
怎么应对这样的人？
首先他们没有自己的计划和方案，他们只是在你原来方案的做了修改，然后直接剥夺了你的成就感。
其次他们中的大多数会认为既然负责监理这个CASE，如果不说点什么就会失去了存在感。
一旦碰到这样的客户，本来用一小时就可以完成的项目，很有可能就要反反复复的做一上午，浪费时间不说，还会影响到工作的心情。（注：一般当客户让我改到第三版以后，我会很烦燥，就开始糊弄他们了。）
如果遇到这样的一个上司或者客户你怎么办？奋起抗争？还是息事宁人任由其蹂躏？
我们副总一般会选择避重就轻的做法：
一般就给客户出三个方案，如果客户提出要修改，有时间的话会改一下，毕竟是客户么，如果没时间就直接搁置到对方不能再等，选一个方案不修改或者稍作修改后完工。
人，关键还是人，事情好做，难就难在你是在跟人打交道，所以不论做什么都要留个心眼。
身边也发生过这样的事情：客户想要一个设计方案，自己公司的设计师拿不出好的来，于是负责人找了其他公司的设计师来做，那边做完了设计方案发过来以后，负责人说不符合我们的要求没有通过直接pass掉了对方的方案，让自己的设计师照着改一些交给了客户。
职场上的话，是要拆开了听的，盲目做事的结果最终只会导致你出力不讨好。
在做事之前，在接到委任的时候请务必要询问好你的领导或者客户的要求或者意见，如果对方没有给出，那就跟对方提出若要更改可以接受的条件，做出的方案不要太多，给自己留点保留的余地。
子之熊掌，彼之砒霜，你不是客户肚里的蛔虫，你永远不能知道一个什么信息都不透露给你的客户想要从你这得到什么样的产品和服务，也没有谁可以实现让每一为客户满意的承诺。如果你是一个新手，那你少不了被你的客户或者主管这样折腾，   在同样付出了努力和心血之后，好与坏，完美和残次，也不过就只是客户或者老总的一句话而已，不要因为别人的评价而妄自菲薄，或许，这只是做事的方式有点问题而已。
人在江湖漂，哪能不挨刀。
每一次挨刀都是一次历练，每一次疼痛都是收获。
还能挨刀，说明你对工作还有激情；总是挨刀，说明你还不懂工作。
