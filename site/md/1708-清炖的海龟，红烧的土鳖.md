# 清炖的海龟，红烧的土鳖 

新年聚餐，主厨的大名叫“VC”，菜谱是“清炖海龟”和“红烧土鳖”……喂喂喂，不要流口水，这顿饭可不是免费的啊，金融海啸银根收紧，麻烦各位自掏腰包：海龟8888元一斤，土鳖88.88元一斤，请问你想吃海龟还是吃土鳖？
10年前，这个问题的答案很明确。你如果有个国外的学位，只要不至于是索马里或者斐济岛的电大文凭，必定有人会三顾茅庐来高薪请你。
10年前，优质土鳖严重缺货。开一家高科技公司，员工还不一定都是大学生，先要电脑培训，教中文输入、浏览网页、如何使用Office。那时候跳槽是家常便饭，后来大家买房了，随便跳槽的现象才延缓了下来，因为白领阶层是中国房地产行业的衍生品，按揭买房后，换工作势必三思而行，否则下个月房子的按揭款没有了着落怎么办？
2000年前后，中国大地掀起了一场海龟回游创业热潮。在北京的嘉里中心和上海中信泰富的楼顶上，海龟们纵火焚烧了VC好几个亿的美元……这件海龟放火烧钱的伟大创举，把刚从蛋壳里爬出来小土鳖们给看得口瞠目呆，影响是深远的。
2005年左右，土鳖们开始成熟上市了。例如，中国移动电信行业中，上千家SP当中，只有寥寥几只海龟，绝大多数是土鳖团队。土鳖生不逢时，没VC给钱烧，只好自己去觅食……大家一哄而上，用短信、彩信、WAP、IVR、抽奖、八卦、群发……把几亿老百姓的手机给搞火爆了，在老大中移动发威之前，差不多个个赚得盆满钵满！
Web 1.0的时候，海龟唱的是主角。那时候海龟创业动不动就能吃到第一口水，美国有Yahoo、eBay、Amazon，海龟只要瞄上一眼，回到中国依样画瓢马上复制一个，VC就立刻扑过来追捧。
Web 2.0的时候，土鳖集体出场共唱“同一首歌”，美国有MySpace，YouTube、Facebook，一旦被土鳖瞄上一眼，第二天会像雨后春笋那样冒出几十家“Me Too!”山寨，一时间把VC都看花了眼，打不定主意到底应该投哪家?
金融海啸，时令不佳，这海龟和土鳖的故事该如何继续下去？要不，咱们来给海龟和土鳖们算算二者来年的运程，提几点新年祝愿？
物以稀为贵，先来看海龟：
(A)“第二次留学”
因为有了互联网，有了北京奥运会，有这么多盗版软件和DVD，中国和世界的距离在缩短，地球扁平了。今非昔比，今天，如果海龟们在回游到祖国的怀抱之前腰包里还没有搞到VC的钞票，如果没有第一手资源和特别的绝招，那么你没资格神气活现地回来做“空降兵”，没法再玩信息不对称的游戏，你得收起海龟的优越感，重新做起，准备“第二次留学”吧。
(B) 适者生存
海龟归来人生地不熟，即使找VC，他们也盯着你是否有水土不服的嫌疑。
中国的市场大得很，但别想当然，别以为跑北京打通部委就等于搞定全中国，那没用，你还得一个省一个省去打点，各个击破，中国的市场还处在“诸侯割据”时代之中……
中国是个效仿者遍地的地方，一旦你有个什么新名堂新花样，大大小小的“Me Too”会来紧紧地簇拥着你，模样和你差不多，价格比你低一半……
中国市场上的产品不仅要求价格低廉、功能齐全，做销售也有中国的一套，要加上递烟、敬酒和三陪，请问海归博士CEO们你们都准备好了吗？
海龟们必须痛苦并快乐着去适应中国现实的商业环境。要回中国混出个眉目来，得走达尔文的道路：适者生存。
(C)  价格本土化
北京、上海的房价、物价和人工成本已经直逼国际大都市水准，别说买房就是租房都得考虑到N环以外。不是说中国有价格优势吗？没错，关键是你从什么角度看，你赚人民币来支撑国际大都市的成本，哪年哪月才能赚到你的第一桶金？
成本本土化，一块人民币就当一块美金花，不要再除以8，扔下去像没感觉似的。中国是价格特别敏感的地带，你想略微提高一下你的利润标准，当心订单就会落入你的竞争对手手里。
工资本土化、成本本土化、价格本土化，海龟需要大力提升自己的性价比。
(D)  二线三线城市的机会
仔细看一看地图，不要对北京、上海情有独钟，那里哪像中国？尝试走一遍长征的道路，从深圳出发，去东莞、再去温州、杭州、苏州、无锡、武汉、西安、成都……最后看看银川、青海、延安。那时再下结论，哪里更有机会？哪里更像中国？
(E)  别痴迷纯技术、多做应用
白猫黑猫，抓住老鼠才是好猫，赚钱才是硬道理。
中国是个“短平快”的战场，别蹲在实验室里磨你的新技术新发明，别做“完美主义者”。这里需要的是更多的快速的杀手级应用，你得跳上了汽车就油门踩到底死劲往前冲。
以前人们说，如果你能征服美国，你就可以征服世界。所以日本人索尼、丰田去了美国，韩国人三星、现代去了美国，欧洲人诺基亚、SAP也去了美国……现在这句话升级成2.0版了：“如果你能在中国成功，你才可能征服世界”。
……
土鳖也在变，第一代的有“原罪”的土鳖创业者，出国考察、海外置业、“回程投资”办企业的、当天使的层出不穷；新生代的土鳖创业者也风起云涌……无论是第一次还是第二次创业，土鳖们得再来一次脱胎换骨，看看这回创业是否可以干得更加漂亮。
第一，别学当土皇帝
少读“资治通鉴”，读一点乔布斯、林肯、马丁路德金。别做土皇帝，不然成功的土鳖一旦上了胡润财富榜就得准备公安局来清查……
第二，拿来主义，山寨精神
别拿来了皮毛把精华给拉下了，山寨的大旗上应该写上“品质”、“品牌”、“品德”！
第三，Clean up your books!!
土鳖公司普遍存在的致命软肋，是做账的基本功。为了避税在外走账，声称你公司年收入上亿，净利润过半，一审计，大家汗颜。投资者、收购者，都到了准备掏腰包的时候，又都把手缩了回去，一场空欢喜，你说冤不冤？
土鳖们要想拿到国际机构投资人的钱，要想做大，要想被世界500强收购，要想上市……一定得有本干净的帐！
第四，别自卑
好笑的是，很多土鳖明明自己干得不错，却凡事便喜欢说“人家美国人如何如何”，其实自己连一趟美国都还没有去过！等你赚了大钱，也去世界逛逛。
第五，中国2.0
  赚血汗钱的时代是旧社会。中国需要干净的空气、没有污染的河流，需要保护自己的自然资源，需要无毒的奶粉、高质量的商品、诚实的服务承诺……你得时刻想到为子孙后代造福，别当历史罪人。
……
因为有了海龟和土鳖，中餐四大菜系中又多了两道极品佳肴；因为有了海龟、土鳖，世界变得扁平了，中国也和世界更靠近了。
海龟 + 土鳖，将同台演出中国创业的2.0：
 *  海龟和土鳖的价格将趋同，海龟和土鳖将在同一口大锅里一起煮；
 *  竞争将不再是学历、语言能力，而是赚钱的本领；
 *  放养的地域不再限于北京上海，将扩散到二三线城市、中西部地区；
 *  走出IT圈；去服务业、品牌、环保再生、新农业、文化创意行业……
 *  手里握的不再仅仅是美钞，还多了人民币；
 *  模仿不再令人青睐，创新、核心竞争力、品牌、知识产权将成为主攻战场；
 *  国际的品质，中国的价格；
 * 如果在中国市场上打得赢，就具有国际市场的竞争力；
 *  海龟 + 土鳖，世界级的伟大中国企业将横空出世……
开饭啦，翠花上菜喽……清炖海龟、红烧土鳖……大家元宵节快乐！
