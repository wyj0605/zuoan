再制作（技术向）

可从 zuoan.sql 恢复数据，导出为 Markdown/HTML 后生成电子书。

# 使用 Pandoc 生成 EPUB
pandoc input.md -o 左岸读书.epub --metadata title="左岸读书" --toc

# 使用 Calibre 生成 EPUB 或 MOBI
ebook-convert input.html 左岸读书.epub --language zh --title "左岸读书" --embed-all-fonts
ebook-convert input.html 左岸读书.mobi --language zh --title "左岸读书" --embed-all-fonts
