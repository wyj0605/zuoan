用免费软件装出完美电脑

2026-02-08 18:20:55

微软“黑屏”事件对中国用户影响巨大，就连中国国家版权局日前也出面对这一事件进行了表态，委婉提醒微软考虑发达国家和不发达国家的经济水平差距，根据中国国情考虑其产品的定价问题。

随着软件产业的发展，在高价政策不得人心的同时，免费软件占领了越来越多的阵地，其实用免费软件也能装备出功能强大的电脑。

●杀毒厂家替用户打补丁

上次，为了电脑不被“黑屏”，许多用户关闭了微软操作系统的自动更新功能，转而使用其他软件公司提供的“打补丁”软件。来自360安全中心的统计数字显示，10月中旬传出微软“黑屏”计划以来，全国有上千万网友关闭了系统自动更新，在不到两周的时间内，国内网友使用360安全卫士软件“打补丁”的总次数超过了5亿次。

为了吸引盗版XP系统用户的注意，提高自己在杀毒和修补系统漏洞能力方面的影响力，国内的杀毒厂家开始借此机会大力宣传自己。金山公司在微软“黑屏”计划实施后立即发表声明，称“金山清理专家”公布的漏洞补丁都是经过金山全球反病毒应急处理中心严格测试、真真正正与系统安全有关的补丁。“金山清理专家”不会自动在用户的计算机上更新安装微软正版验证补丁。也就是说，关闭了电脑系统自动更新的用户，用金山清理专家安装系统漏洞补丁就可以躲避“黑屏”问题。 360安全卫士也告诉用户，用自己的漏洞修补工具，可以帮助用户下载那些用于系统安全的补丁。

关闭系统自动更新的用户，用第三方杀毒厂商提供的漏洞修补工具是个不错的选择，第三方的打补丁工具里有些会帮用户选择，譬如只选那些用户真正需要的安全类补丁，而用户不需要的，例如会导致‘黑屏’的微软正版验证补丁，就不会让用户打。这是不是中国人的上有政策，下有对策的思想呢？

●免费软件装出强劲电脑

如果我们没有这些高价的软件，那我们的电脑就只能处于瘫痪状态吗？其实，我们有很多的选择，这里包括我们使用习惯的选择。大家如果有心，足可以用免费的软件打造出一个完整的电脑软件系统。

1.操作系统——Linux

在操作系统方面，微软Windows系列操作系统早已成为霸主，要想舍弃并不容易，如果要进行替换，国产的开源Linux或许是个不错的选择。Linux 操作系统的正版软件在开源的官方网站可以免费下载。Linux不仅“体积”小，而且运行非常稳定。现在，很多Windows应用程序都可以运行在 Linux环境下，速度和性能与运行在Windows环境下基本没有差别。

2.办 公 软 件 ——WPSOffice或OpenOffice

经常用电脑的人，都会使用到办公软件，使用最广泛的无疑是微软的Office系列。正版微软Office办公软件价格不菲，我们可以选择国产金山软件刚刚推出的WPSOffice个人永久免费版办公软件。另外，OpenOffice也是不错的备选。

3.杀毒软件——360安全卫士或Avira

如果电脑操作系统是Linux的，基本不用任何杀毒软件就能横行天下，但在Windows操作系统下，杀毒软件是必不可少的。目前杀毒软件市场竞争激烈，有些国产杀毒软件推出了免费使用半年或一年的口号来吸引用户，用户可以选择这些国产软件。不过，完全免费的防毒软件推荐360安全卫士。

Avira AntiVir Personal Edition Classic是一款德国的免费杀毒软件，小巧但杀毒能力强大，免费版本的功能就比较全面了，可以应付大部分用户的防木马病毒的需求。

4.图形图像处理——光影魔术手GIMP （免费版）

很多用盗版Photoshop图像处理软件的用户可能不知道，Photoshop的正版售价高达4000元以上，个人用户肯定无法承受。替换软件推荐光影魔术手（免费版），它简单、易用，不需要任何专业的图像技术，就可以制作出专业胶片摄影的色彩效果，是用于摄影作品后期处理、图片快速美容、数码照片冲印整理不错的图像处理软件。

5.图片浏览软件——Picasa

基本上，图片浏览软件用系统自带的就行，但如今使用最广泛的图片浏览软件ACDsee也要收费，因此建议用Google公司的Picasa软件替代。 Picasa原为独立收费的软件，功能非常强大，被Google收购后已经变成免费软件，浏览图片速度非常快，并具有一定的图片编辑修改功能。

6.压缩和解压软件——7zip

WinRAR是众多用户平常在交流资料时的必备工具之一，但也需要付出一定的使用费。7zip这款软件或许并不为人所熟悉，但它是一款完全免费的压缩工具，可以打开包括RAR/ZIP/GZIP在内的常见压缩格式，是一个替代WinRAR的不错选择。

7.代码编辑：Notepad++

Notepad++是一款完全免费的源程序代码、HTML网页代码编辑软件，支持多达数十种常见源代码或脚本的语法，包括C，C++，Java，C#，XML，HTML，PHP，Javascript，RC resource file，makefile，ASCII，doxygen，ini file，batch file，ASP ，VB/VBS，SQL，Objective-C，CSS，Pascal，Perl，Python，Lua等，功能非常强大。

8.WEB服务器：Linux + Apache + MySQL + PHP

替换目标：Windows + IIS + SQLServer + ASP

LAMP（Linux + Apache + MySQL + PHP）是完全免费开源的服务器架构，具有简易性、低成本和执行灵活等特点，具有超过Windows + IIS + SQLServer + ASP的性能，并且完全免费，是搭建WEB服务的首选架构。

以上月光博客介绍的十款功能强大的免费软件，这些免费软件大部分都支持中文语言，并且全部都没有插件，在功能上丝毫不逊色于正版的商业软件，是替代破解和盗版软件的推荐选择。
