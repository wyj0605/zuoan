立春至，用希望怀抱你我

2026-02-11 14:13:49

文/韦宇教2015 年2 月 4日，立春。古籍《群芳谱》对立春作了如下定义：“立，始建也。春气始而建立也。”“一候东风解冻，二候蜇虫始振，三候鱼陟负冰。”立春，是世间万物一切希望的开始。

过了立春，白昼变长，太阳回暖，万物复苏，柳绿花开。春姑娘一来，似乎一切都变得美好了起来，连同北京的雾霾都会慢慢消散掉。古语有云：“年龄不饶人，节令不饶天”。立春一到，春天的大门好像一夜之间就敞开怀抱，迎风送暖。安定门西滨河岸边的柳条，在微风中已经染上了一层层新绿，柳枝上也鼓出了一个个芽尖。

木制的长廊上，有老人在散步，有小孩在嬉戏，有情侣在海誓山盟，有梦碎者在思考人生。我来回穿梭其中，像个旁观者，沉默不语，也辨不清他们内心的喜怒哀乐，是否真如同他们的表情所展露出的一样。我每天上下班都会路过这里，从立春走到立夏，从立秋走到立冬，再走回到立春，循环往复，轮回开始。在这里，我总能看到很多人，不管是步行的，骑车的，还是开车的，打车的，总是行色匆匆，一如你我。一到“白雪却嫌春色晚，故穿庭树作飞花”的时节，我总是忍不住想起她，国内第一个最出名的女贝斯手 ——筠子。

她在高晓松作词、朴树作曲的《春分》里唱到：“谁听见海里面 / 四季怎样变迁 /谁又能掀起那页诗篇 / 谁能唱/谁能让怀念停留在那一天永不改变……来啊 / 来看那春天/ 她只有一次啊 / 而秋天是假的 /生活多遥远啊 / 你不要/ 不要脱下冬的衣裳 /你可知/ 春天如此短 / 她一去就不再来”。应了歌中所言，筠子真的“她一去就不再来”，如流星一样，闪耀却瞬间坠落，定格在 23 岁的青春。筠子在录《春分立秋冬至》这张专辑时，她说她并不快乐，甚至是非常忧虑的。在她很喜欢的《青春》这首歌里，她这样唱着：我心里什么都没有 /就像没有痛苦 /这个世界什么都有 / 就像每个人都拥有。在充满朝气和希望的立春，想到这些，似乎不合时宜。

但是，我坚信，筠子只是想以这样一种方式来保持自己内心的完整，让自己的生命停留在最灿烂的时刻，让青春让梦想盛开在永不凋零的年华里。虽然春分是立春过后的第四个节气，虽然《春分》这首歌并不是描写立春，但在太阳到达黄经 315度的今天，我还是会不由自主的想到了筠子，想到她酷似王菲的声线，想到那个真实律动的生命。她的青春虽然已经消逝，但梦想不减，希望不灭。然后，我抬起头，仰望着这个城市的上空，心里哼唱着汪峰写给筠子的《美丽世界的孤儿》 —— 你看车辆穿梭 /远处霓虹闪烁 / 这多象我们的梦。活着真好，至少我们还有机会，可以背起行囊，去追寻彼岸的风景，彼岸的梦想，彼岸的幸福。

在单曲循环《春分》这首歌，并写下这些文字的时候，我脑海里还会同步浮现出顾长卫导演的电影《立春》。这部电影与《春分》有着异曲同工之妙，都是在讲述在遥不可及的理想面前，小人物如何用生命去坚守和坚持内心的梦想田园，哪怕结果一败涂地，哪怕奇迹最终没有发生，但绝无后悔之意。“立春一过，城市里还没有什么春天的迹象，但风真的就不一样了。”这是这部电影开始时的独白，有些感伤，有些悲悯，但又不失希望；有些理想，有些文艺，但又留有遗憾。法国诗人兰波“生活在别处”的穷其一生的口号，成了电影里的主角配角们向往外面世界的枷锁和魔咒，一旦动心，必死无疑。

“看见每一个拎着包离开这个城市的人，我都很羡慕。”他们就在这样的念想里，一边怀抱着纯粹的理想，一边对抗着现实的利刃，在这两股力量的相互厮杀下，期待着有一天，梦想的光芒能够照亮现实的黑暗。看着电影里面六个人演绎的六个故事，看着他们所经历的或逃离或挣脱或抗争或屈服的生活，说不上是同情，还是悲愤，只是感觉很真实，就像我生活了七年的这个城市，就像这个城市里每天挤着地铁上下班的你我他。这个城市，每天都有人怀揣着梦想而来，同样地，每天也都有人带着遗憾而离开。这个城市，每天都有人拿着行李说走就走，却又义无返顾地选择了再次归来。

这个城市，给了很多人真实活过的标注，那些真实里有着他们孤傲的青春、灿烂的年华以及始终未曾忘却的纯真的梦。就像《立春》里的音乐教师王彩玲，美术青年黄四宝和胖子周瑜。我知道，他们是多么不情愿目睹自己青春的热爱就这样老去。我知道，他们以为那些梦想是他们离开那个小城市前一直在做的梦，而如今在坚硬的现实面前，只能变成深楚的念想。一如顾长卫所说：“离理想越来越远，这不是悲剧。

”是的，不管停留在哪个城市，不管选择何种生活方式，生活总是在梦想和现实之间划出一道道深浅不一的痕，我们只能在那道痕内外不停的游走，直到有一天亲手把它抚平。“律回岁晚冰霜少，春到人间草木知。便觉眼前生意满，东风吹水绿参差。”《立春》于你我而言，或许只是一次梦起梦碎的旅程，或许只是一场青春的惦念和祭奠。期间，我们带着成长和逐梦的伤痛以及喜悦，因为那终将是一场成长和生活之殇。
