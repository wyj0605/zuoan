蜗牛背着那重重的壳

讲一个励志故事先。
小蜗牛问妈妈：“我们为什么要背着一个重重的壳？”
“因为我们没有骨头的支撑，只能爬着前进，需要这个壳来保护我们的身体。”
小蜗牛又问：“毛虫姊姊也没有骨头，也要爬，为什么她不用背着一个壳？”
“因为毛虫会变成蝴蝶，她会长出翅膀在天上飞。她有天空的厚爱。”
小蜗牛又问：“蚯蚓哥哥也没有骨头，也要爬，为什么他不用背着一个壳？”
“因为蚯蚓会钻洞，它有大地的厚爱。”
小蜗牛哭了。“为什么天空和大地不爱我们？”
“所以我们有壳啊。”妈妈说，“我们不靠天，不靠地，只有靠我们自己！”
有一天，《赢在中国》的一个选手讲了这个故事，他说，他就是那个蜗牛。
他说，上帝把所有的玩笑，都开在他的相貌上了。他如何不容易，没有天地可以靠，等等……
然后？
俞敏洪说，“我从你的经历中看到你的挣扎，成长，变成精英，但是你太多的在意自己的个性和感受，为什么要觉得别人会鄙视你呢？比如我当初被北大处分 的时候，我也觉得每一个北大人都在鄙视我，其实好多人都根本不知道我是谁！”他分析道，“你内心有一些虚弱，所以才会建一个盔甲，就像你说得蜗牛的壳一 样，是你自己加上去的。”现场观众鼓掌表示认同，“如果你再多点勇气的话，就可以把这个壳去掉，长出一双翅膀，在天空中翱翔。我用了10年的时间，才把自 己的背上的壳去掉，既靠天，又靠地，还靠自己。我觉得你要有这样的大气，如果把刚才的气势拿出来，你一定能做成很大的事情，但前提是要把自己背上的壳去 掉，一定要做到这一点！”
看了蜗牛的故事，觉得很励志，即使没有优越的出生，做人靠自己也可以成功。但看了俞的回答，发现自己身上缺乏这样的魄力，让自己变成可以依靠。有天地可靠，也是一种本领。
励志故事，除了严重地励志，还会严重地误导。蜗牛的故事就是这样。关键是要被励志，而不要被误导。但是，一般人很难分清这两个部分。（其实，绝大多数故事都有这两面。）
小蜗牛的确有一个壳。然后你认为自己是个小蜗牛，你也有个壳。问题是，你不是小蜗牛，你的那个壳，已经被俞敏洪老师英明地指出，是你自己背上去的。
那个选手，自己给自己背上一个壳，集自卑与狂妄于一身。
我们都有自己给自己背上的壳。请把它放下，然后轻装上路，靠自己，但是也要努力靠天靠地。
